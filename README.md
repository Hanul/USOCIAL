# USOCIAL
각종 SNS 서비스와 연동하는 기능들을 담은 UPPERCASE용 BOX입니다.

## 설치하기
1. 프로젝트의 `DEPENDENCY` 파일에 `Hanul/USOCIAL`를 추가합니다.
2. [`ubm`](https://www.npmjs.com/package/ubm)을 이용해 설치합니다.
	```
	ubm install
	```

## 지원하는 SNS 서비스
- Facebook
- Twitter
- Instagram
- Line
